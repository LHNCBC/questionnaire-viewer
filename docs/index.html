FHIR Questionnaire Viewer
Using LHC-Forms Widget

This is a tool that allows you to render a FHIR Questionnaire (JSON), with a FHIR resource package file that provides resources needed by the Questionnaire.

See https://confluence.hl7.org/display/FHIR/NPM+Package+Specification

*Note: These field values can be specified in the URL. Use URL parameter "q" to provide the URL for the Questionnaire JSON, and parameter "p" to provide the URL for the FHIR resource package (package.tgz).
